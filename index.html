Authentication + Setup

Setup

Authenticate with GitHub CLI

1
Open the terminal in VS Code
2
Run gh auth login
3
Choose GitHub.com
4
Choose HTTPS
5
Choose Login with a web browser
6
Complete authentication in the browser
7
Confirm with gh auth status
Setup

Install and Launch Gemini CLI

1
Install Gemini CLIbrew install gemini-cli # macOS npm install -g @google/gemini-cli # Windows
2
Run gemini in your terminal
3
Complete the Google login flow
4
Return to the terminal
5
Confirm Gemini is ready for prompts
Demo

Establish an Agent Workflow

Before making changes, the agent must propose a plan and wait for approval.
"Inspect this folder. Propose a plan before changing anything."
"Do not create, move, or delete files until I approve the plan."
